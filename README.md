# Finance_transactions_datasets

This project focuses on the exploratory data analysis (EDA) and processing of financial transaction datasets. Using Python and Jupyter Notebooks, it identifies patterns, trends, and insights within transaction records to help understand financial behavior or detect anomalies.

## Repository Structure

*   **Task2.ipynb**: The main analysis notebook containing data cleaning, visualization, and transactional insights.
*   **README.md**: Project documentation.

## Features

*   **Data Cleaning**: Handling missing values and formatting currency/date fields for analysis.
*   **Exploratory Data Analysis (EDA)**: Visualizing transaction volumes, categories, and time-based trends.
*   **Insight Generation**: Summary statistics and key performance indicators (KPIs) for the dataset.

## Tech Stack

*   **Language**: Python
*   **Environment**: Jupyter Notebook
*   **Libraries**: Pandas, Matplotlib, Seaborn (standard for financial data manipulation and visualization).
